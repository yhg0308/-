# 🦟 灭蚊大战 — Mosquito Slayer

夏日复古漫画风格的灭蚊游戏，使用 Canvas 2D + Web Audio API 构建。

## 玩法

- 屏幕上始终有 **20只蚊子**，击杀后立即在边缘重生
- **体力系统**：初始 100/100，使用道具体力消耗，每 3 秒恢复 1 点
- **4种道具**：手、苍蝇拍、电蚊拍、蚊香（击杀解锁）
- **3种蚊子飞行模式**：盘旋、冲刺、布朗运动

## 道具

| 道具 | 解锁条件 | 体力消耗 | 操作方式 |
|------|---------|---------|---------|
| 🖐️ 手 | 初始 | -5 | 精准点击 |
| 🪰 苍蝇拍 | 击杀10只 | -2 | 点击（2倍范围） |
| ⚡ 电蚊拍 | 击杀50只 | -1 | 拖拽划过 |
| 🪴 蚊香 | 击杀200只 | -10 | 点击放置（驱赶） |

## 操作

- **鼠标点击/拖拽**：使用当前道具
- **数字键 1-4**：切换道具
- **ESC**：暂停/继续

## 部署

直接使用静态文件服务器托管整个目录，或部署到 GitHub Pages：

```bash
# 本地预览
npx serve mosquito-game/
```

## 文件结构

```
mosquito-game/
├── index.html          # 主入口
├── css/style.css       # 样式
├── js/
│   ├── main.js         # 游戏循环与状态机
│   ├── mosquito.js     # 蚊子类
│   ├── player.js       # 体力与击杀
│   ├── tools.js        # 道具系统
│   ├── particles.js    # 粒子特效
│   ├── audio.js        # 音频系统
│   └── leaderboard.js  # 排行榜
└── assets/             # 图片与音频素材
```

## 技术栈

- Canvas 2D 渲染（requestAnimationFrame 循环）
- Web Audio API（合成嗡嗡声 + 立体声空间化）
- LocalStorage 排行榜
- 响应式缩放（设计稿 1920×1080）
